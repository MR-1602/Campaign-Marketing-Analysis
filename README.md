# 📊 Marketing Campaign Analysis Dashboard

This project analyzes marketing campaign performance using Power BI.  
The dashboard provides insights into customer engagement, campaign effectiveness, and revenue contribution to help businesses optimize marketing strategies.

---

## 🚀 Project Overview

Marketing teams run multiple campaigns across different channels, but it can be difficult to measure which campaigns deliver the best results.

This project analyzes marketing campaign data to answer key business questions such as:

- Which marketing campaigns generate the highest revenue?
- Which customer segments respond best to campaigns?
- What channels drive the most engagement and conversions?
- How marketing performance changes over time

The interactive dashboard helps stakeholders quickly identify high-performing campaigns and optimize future marketing strategies.

---

## 🛠 Tools & Technologies

- **Power BI** – Data visualization and dashboard creation
- **Power Query** – Data transformation and cleaning
- **DAX** – Calculated measures and KPIs
- **Data Analytics** – Marketing performance analysis

---

## 📈 Key Dashboard Metrics

The dashboard tracks important marketing KPIs including:

- Total Revenue
- Campaign Conversions
- Customer Engagement
- Campaign ROI
- Revenue by Campaign
- Revenue by Marketing Channel
- Customer Segmentation

---

## 💡 Key Insights

The dashboard helps identify:

- Top-performing marketing campaigns
- Most profitable customer segments
- Marketing channels with the highest ROI
- Trends in customer engagement and campaign success

These insights support data-driven decision making for marketing teams.
